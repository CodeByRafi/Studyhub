# StudyHub

## Goal
A platform for university students to browse and upload notes, questions, research papers, jobs, mentorship, and AI tools.

## Tech Stack
- Frontend: Next.js + TypeScript + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL + Prisma

## Build Order
1. Frontend setup
2. Backend setup
3. Database setup
4. Auth
5. Dashboard
6. Notes system
7. Comments + Ratings
8. Jobs + Mentorship
9. AI tools
